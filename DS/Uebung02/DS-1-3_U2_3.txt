    value           |   logical shift   |   arithmetic shift
--------------------|-------------------|--------------------
 int8_t x = 0xAA    |                   |
    x >> 2          |        2A         |          EA       
    x << 3          |        50         |          50

(AA)_16 = (10101010)_2

logical shift:
(10101010)_2 >> 2 = (00101010)_2 = (2A)_16
(10101010)_2 << 3 = (01010000)_2 = (50)_16

arithmetic shift
(10101010)_2 >> 2 = (11101010)_2 = (EA)_16
(10101010)_2 << 3 = (01010000)_2 = (50)_16
